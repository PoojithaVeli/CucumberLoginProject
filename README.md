# CucumberLoginProject
# Cucumber Selenium Login Automation Project

## Tools Used
- Java
- Eclipse
- Maven
- Selenium WebDriver
- Cucumber
- JUnit

## Features
- Login Feature File (Gherkin)
- Step Definitions
- Hooks (Before and After Execution)
- JUnit Test Runner
- HTML Cucumber Report

## Execution
Run TestRunner.java as JUnit Test.
